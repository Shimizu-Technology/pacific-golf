# Pacific Golf — Branding Presets

Use these presets as a fast starting point so every client site feels intentional and distinct.

---

## 1) Commemorative Preset (Airport / Legacy / Honor)
**Best for:** Government orgs, memorial tournaments, anniversary events.

### Visual Direction
- Primary: deep navy (`#1e3a5f`)
- Secondary: steel blue (`#2c5282`)
- Accent: ceremonial gold (`#c9a227`)
- Background: light gradient with subtle texture
- Typography: strong sans for titles + elegant serif accent line (optional)

### Layout Priorities
1. Official logo lockup at top
2. Event + edition hierarchy (e.g., "5th Annual")
3. Honoree/memorial marker (silhouette/badge)
4. Clean event details card
5. Strong single CTA

### Signature Modules
- Hero with logo + annual title stack
- Memorial/Honoree visual lockup
- Formal contact section

### Tone
Formal, respectful, official.

---

## 2) Mission/Charity Preset (Make-A-Wish / Cause-Based)
**Best for:** Nonprofits, fundraising events, community impact tournaments.

### Visual Direction
- Primary: charity brand blue (org-specific)
- Secondary: white/neutral with high readability
- Accent: hopeful/energetic accent (optional yellow/teal)
- Background: clean, airy, approachable
- Typography: highly readable sans-serif

### Layout Priorities
1. Mission-led hero copy
2. Event details + easy registration CTA
3. Sponsor trust signals
4. Impact storytelling modules

### Signature Modules
- Mission statement block
- Impact module (wishes granted / families served)
- Sponsor wall by tier
- Optional testimonials/story cards

### Tone
Warm, trustworthy, community-focused.

---

## 3) Competitive/Sport Preset (Tournament / Performance)
**Best for:** Sports-first events where live action + leaderboard are core.

### Visual Direction
- Primary: strong contrast palette (dark + accent)
- Secondary: neutral slate background layers
- Accent: high-visibility action color
- Background: modern, high-contrast sections
- Typography: bold headings, compact stat text

### Layout Priorities
1. Event state/status (open/live/final)
2. Fast access to registration and leaderboard
3. Live score/raffle visibility
4. Mobile-first information density

### Signature Modules
- Live status ribbon
- Leaderboard teaser block
- Scoring CTA / golfer access module
- Raffle panel and sponsor highlights

### Tone
Energetic, direct, action-oriented.
